# Xillen C2 Framework

Фреймворк для управления и контроля над агентами в инфраструктуре Red Team.

## Возможности
- Управление агентами
- Командование и контроль (C2)

## Быстрый старт
```bash
go run main.go
```

## Требования
- Go 1.18+
